CentOS环境JDK11安装

1、下载

JDK11下载地址：https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html

2、路径

创建jdk安装文件夹

mkdir /usr/local/java

3、安装

进入下载的jdk所在目录，执行安装命令。

tar -zxvf jdk-11.0.10_linux-x64_bin.tar.gz -C /usr/local/java/

4、环境配置

修改系统环境文件/etc/profile，在最后加上java环境变量配置信息。

export JAVA_HOME=/usr/local/java/jdk-11.0.10
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

5、连接

创建软连接

ln -s /usr/local/java/jdk-11.0.10/bin/java /usr/bin/java

6、验证

java -version

输出结果：

java version "11.0.10" 2021-01-19 LTS
Java(TM) SE Runtime Environment 18.9 (build 11.0.10+8-LTS-162)
Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.10+8-LTS-162, mixed mode)
