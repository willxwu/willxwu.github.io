Docker安装PHP

  1. 拉取PHP镜像

    docker pull php:7.4-fpm

  2. 运输PHP容器

    docker run -d -v /mydata/nginx/html:/var/www/html -p 9000:9000 --link mysql:mysql --name php php:7.4-fpm

  3. 进入PHP容器

    docker exec -it php /bin/bash

  4. PHP扩展安装(非必须)

    docker-php-ext-install pdo_mysql

  5. 设置随Docke启动

    docker update --restart=always php

注意：自己映射路径/mydata/nginx/html。
