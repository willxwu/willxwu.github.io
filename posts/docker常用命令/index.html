Docker常用命令

一、镜像

  1. 查看docker容器镜像

    docker images

  2. 拉去容器镜像

    docker pull 容器

  3. 删除容器镜像

    docker rmi 容器

二、容器

  1. 安装容器，-d后台运行，-v映射路径, -name容器名称, -link链接外部容器，-p 端口号映射，-e设置密码

    docker run -d -v /mydata/nginx/html:/var/www/html -p 9000:9000 --link mysql:mysql --name php php:7.4-fpm

  2. 启动容器

    docker start 容器

  3. 关闭容器

    docker stop 容器

  4. 重启容器

    docker restart 容器

  5. 查看运行容器

    docker ps

  6. 查看所有容器

    docker ps -s

  7. 删除容器

    docker rm 容器

  8. 查询当前容器

    docker container ls -all

  9. 删除未启动容器映射

    docker container rm 容器

  10. 查看容器映射信息

    docker inspect 容器 | grep Mounts -A 20

  11. 进入容器内部

    docker exec -it 容器 /bin/bash

  12. 设置容器随Docker启动

    docker update --restart=always 容器

更多命令参考Docker官网
